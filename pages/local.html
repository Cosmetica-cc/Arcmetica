You've found Arcmetica!

And you're selfhosting? Cool bro.

Arcmetica works by redirecting requests from OptiFine's cosmetics server to our own. This method lets us show drippy cosmetics in almost any Minecraft version that has OptiFine installed!

To check if Arcmetica is installed on this computer, click here. If you get a blank page that says "Not found", then Arcmetica isn't installed and you should follow the instructions here (but since you're selfhosting, replace the IP addresses in the tutorial with the IP address of this server).
